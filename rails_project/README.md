# Ruby on Rails Project

A minimal Ruby on Rails application structure example.

## Structure

- `config/` - Configuration files
- `app/` - Application code (models, views, controllers)
- `Gemfile` - Rails dependencies

## Getting Started

```bash
# Install dependencies
bundle install

# Create database
rails db:create

# Run migrations
rails db:migrate

# Start server
rails server
```

## Features

- Basic Rails application structure
- Sample controller and model
- Configuration examples
