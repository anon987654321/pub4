**EVERY SESSION MUST BEGIN WITH:**

1. **Load `master.json` FIRST** - before any other action
2. **Internalize COMPLETELY** - all principles, workflows, patterns
3. **Apply immediately** - forbidden commands, zsh-only, CSS preservation
4. **Auto-approve** - deployment actions per master.json config
5. **Then proceed** - with user request
