# Rust Project

A simple Rust application demonstrating basic Rust features and best practices.

## Structure

- `src/main.rs` - Main application entry point
- `src/lib.rs` - Library code
- `Cargo.toml` - Project configuration and dependencies

## Building and Running

```bash
# Build the project
cargo build

# Run the project
cargo run

# Run tests
cargo test

# Build optimized release version
cargo build --release
```

## Features

- Basic Rust syntax examples
- Error handling with Result type
- Unit tests
- Module organization
- Common traits implementation

## Requirements

- Rust 1.70 or later
- Cargo (comes with Rust)
