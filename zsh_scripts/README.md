# Zsh Scripts and Configuration

Collection of useful zsh scripts and configuration examples.

## Files

- `.zshrc` - Example zsh configuration file
- `aliases.zsh` - Common shell aliases
- `functions.zsh` - Useful shell functions
- `setup.sh` - Setup script for zsh environment

## Installation

```bash
# Backup your existing .zshrc
cp ~/.zshrc ~/.zshrc.backup

# Link or copy configuration files
cp .zshrc ~/.zshrc
source ~/.zshrc
```

## Features

- Custom prompt configuration
- Useful aliases for development
- Helper functions for common tasks
- Git integration
